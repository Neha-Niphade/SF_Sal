# SF Salaries Analysis 

### Description

This repository contains a Jupyter Notebook analyzing San Francisco Salaries data. The notebook demonstrates data manipulation, cleaning, and exploratory data analysis (EDA) using Python, Pandas, and NumPy.

The goal is to explore salary patterns in San Francisco's public sector and gain hands-on experience with data analysis techniques.

### Contents

Loading and inspecting the dataset (SF_Salaries.csv)

Checking for missing values and cleaning data

Basic statistical analysis: mean, median, min, max

Filtering and sorting data

Aggregations and group-bys to analyze trends

Optional visualizations (if added)

### Tech Stack

Python 3.x

Jupyter Notebook

Pandas

NumPy

### How to Use

Clone the repository:

git clone <your-repo-link>


Navigate into the project folder:

cd SF_Salaries_Analysis


Open the Jupyter Notebook:

jupyter notebook SF_Salaries.ipynb


Run the notebook cells to explore the analysis and outputs.
